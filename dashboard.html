❓ How do I get my Client ID & Secret?
🛍 Create an Ecom Rise app on your Shopify store
Step 1 — Open Develop Apps
In your Shopify Admin, go to:
Settings → Apps and sales channels → Develop apps
Click "Build apps in Dev Dashboard"
(This opens the Shopify Dev Dashboard — no separate account needed, it uses your store login)
Step 2 — Create a New App
In the Dev Dashboard, click "Create an app" → "Create app manually"
App name: Ecom Rise
Click "Create app"
Step 3 — Set API Permissions
Go to "Configuration" tab
Under Admin API access scopes, search for and add these permissions:
read_orders read_products read_customers read_inventory
Click "Save"
Step 4 — Copy Client ID & Client Secret
Go to the "API credentials" tab
Copy your Client ID and Client Secret
⚠️ Keep the Client Secret safe — treat it like a password
Step 5 — Install the App on Your Store
Go to the "Distribution" tab
Select "Custom distribution" → choose your store → click "Install"
Approve the permissions when prompted
Step 6 — Paste & Connect
Paste the Client ID and Client Secret into the fields above
Click Test Connection — done! ✅
▸ I already have an Admin API access token (legacy app)
Paste your existing access token if you have one from a legacy custom app.
